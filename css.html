Data & Software Engineering

Program RevoU Tech Academy (Data and Software Engineering) memiliki dua modul kunci yang harus dipelajari oleh setiap mahasiswa, yaitu Software Engineering dan Data Analytics. Agar para peserta mampu memahami keempat posisi kunci ini, secara umum program ini akan dibagi menjadi 4 bagian, yaitu:

  1. Orientation Week (Communication skills, Career in digital economy, Continuous learning, and more)
  2. Software Engineering (Front end, HTML, CSS, Javascript and more)
  3. Data Analytics (Data analysis method, Data Visualization, and more)
  4. Career development (CV & LinkedIn Creation, Interview hacks, and more)
